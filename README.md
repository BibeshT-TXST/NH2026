## Getting Started

### 1. Prerequisite: Gemini API Key
To power the Emotional Orchestrator, you'll need a Google Gemini API Key:
1.  Go to [Google AI Studio](https://aistudio.google.com/api-keys?project=gen-lang-client-0771680053).
2.  Generate a new API key.

### 2. Backend Setup
```bash
cd Backend
npm install
```
Create a `.env` file in the `Backend` directory and add your key:
```env
GEMINI_API_KEY=your_api_key_here
PORT=3001
```
Start the backend:
```bash
npm run dev
```

### 3. Frontend Setup
```bash
cd Frontend
npm install
npm run dev
```

The application will be running at `http://localhost:5173`.
